## 1. 서비스 한 줄 요약

> 직관을 혼자 가는 팬들을, 나와 잘 맞는 성격 & 응원 스타일의 팬과 매칭해주는 동행 매칭 서비스

- 대상: 정관장 여자 배구 팬 (대전 기준)
- 목적:
  - 혼직 → 동행 직관
  - 찐팬 인증 + 직관 기록 + 뒷풀이/카풀까지 이어지는 커뮤니티

---

## 2. 핵심 사용자 시나리오 (정관장 팬 1명 기준)

1. 회원가입
   - 닉네임, 성별(남/여 필수), 연령대, 응원팀 = 정관장(MVP1), 성격/응원 스타일 선택
2. 경기 일정 확인
   - “정관장 경기 일정” 리스트 확인
   - 보고 싶은 경기 상세 페이지 진입 (날짜/시간/장소/상대, 홈/어웨이)
3. 동행 찾기
   - 해당 경기의 **직관 동행 게시판**에서
     - 본인이 글 작성 or 프로필 카드 등록
       - E구역 100블럭 근처 / 20대 남 / #조용히집중 / 직관 2회차 / “처음 글 남겨요~”
     - 이미 올라온 글 리스트 or 프로필 카드에서 마음에 드는 사람 선택
4. 프로필 확인 & DM
   - 상대 프로필: 사진, 성격 태그, 직관 횟수, 찐팬 인증 여부, 매너온도 등 확인
   - DM으로 시간/만날 장소 조율
5. 직관 인증 & 찐팬 성장
   - 경기 당일 티켓 인증 → 직관 횟수 +1
   - 승/패 결과 연결해서 승률 기록
   - 일정 기준 이상일 때 “승리 요정”, “패배의 아이콘” 같은 칭호 부여

## 3. 기능 구조 정리

### 3-1. 회원가입 / 로그인

- 필수 정보
  - 성별 (남 / 여)
  - 응원팀: MVP에서는 **정관장** 고정
  - 연령대/직업: “20대 초반”, “30대 직장인” 등 간단 선택
  - 성격/응원 스타일 태그
    - #조용히집중, #열정응원, #촬영러, #먹방러
    - 뉴비 팬 vs 찐팬
- 목적:
  → 나중에 매칭 시 “비슷한 스타일끼리 / 원하는 스타일끼리” 매칭에 활용

---

### 3-2. 메인 페이지

- 내용
  - 오늘 / 이번 주 **정관장 경기 일정** (나중엔 다른 팀/종목 확장)
  - 경기 상태: 예정 / 진행중 / 종료
- MVP에서는:
  - 리스트 + 간단한 필터(날짜 기준) 정도만 있어도 충분

---

### 3-3. 경기 상세 페이지

- 두 팀 정보
  - 정관장 / 상대팀
  - 순위, 승/패 기록
  - 주요 선수 / 포지션
- 경기 정보
  - 시간, 장소, 홈/어웨이, 좌석 구역 안내
- 부가 기능
  - **응원톡** (경기별 채팅방)
  - **직관 동행** (이 경기 기준으로 동행 게시판 연결)
  - 관전 포인트 (나중에 AI 요약으로 확장)

MVP1에선:

- 기본 정보 + “직관 동행 게시판으로 이동” 버튼만 있어도 OK
- 응원톡/AI관전 포인트는 Phase 2로 빼도 됨

---

### 3-4. 직관 동행 페이지 (핵심 기능)

- 흐름
  1. 경기 리스트 → 특정 경기 선택
  2. 해당 경기의 **동행 방 리스트** 접속
  3. (선택) **방 만들기** 또는 **방 입장 신청**
  4. **승인 시 방 초대 → 방 채팅 참여**
- 기능
  - 매칭(방 기반)
    - **방 만들기(호스트/방장)**
      - 예: **E구역 100블럭 근처 조용직관 방**
        - E구역 100블럭 근처 / 최대 4명 / **승인제**
        - 호스트: 20대 남 / 직관 2회차 / #조용히집중 #초보환영
        - 소개: “처음 방 열어요! 조용히 경기 보실 분만 🙏”
      - 예: **같이 예매/동행 조율 방**
        - 같이 예매 구함(좌석 상의) / 최대 3명 / **승인제**
        - 호스트: 30대 여 / 직관 10회차 / #열정응원 #승리요정 #예매가능
        - 소개: “예매 도와드려요. 응원 같이 빡세게 가실 분!”
    - **입장 신청(=줄서기 느낌 대체)**
      - 유저가 방을 고르고 “입장 신청”
      - 자동 첨부: 내 프로필 요약(연령대/성별 선택/직관 n회차/응원 태그)
      - 한 줄 메시지: “혼자 가요, 조용히 볼게요!” / “열응원 가능, 2인입니다!”
      - 방장은 신청자 리스트에서 **승인/거절/차단**
      - 승인되면 **자동 초대 → 방 채팅 입장**
  - 필터(방 리스트 필터로 이동)
    - 예매 상태(예매완료/예매예정/같이예매)
    - 좌석 구역(블루존/R석 등)
    - 성별/연령대(선택 정보 기반)
    - 응원 스타일(#조용히집중/#열정응원 등)
    - **입장 방식(승인제/자유입장)**, **현재 인원/정원**, 마감 여부
    - 지역
  - 매칭 방식(1:1 DM 제거/축소)
    - 상대 프로필 확인(호스트/참여자 요약)
    - **방 입장 신청 → 승인 → 방 채팅**으로 조율
  - **방 관리 (사이드바)** (**New**)
    - 방장은 사이드바 메뉴를 통해 **공지사항 수정, 참여자 강퇴, 모집 마감, 대기 승인** 가능
    - 참여자는 참여자 목록 확인 및 방 나가기 가능
- 수익 모델 아이디어(mvp1에서 x)
  - “990원 내고 볼 수 있는 깊은 프로필” → **“방 신뢰/매칭 보조”로 연결**
    - 굿즈 보유 리스트
    - 총 직관 횟수 / 승률
    - 매너온도(방 참여/후기 기반 점수)
    - (추가로 잘 맞는 방향) **방 부스트/상단 노출**, **호스트 인증 배지**
      → 이건 MVP에선 안 넣어도 되고, **v2 수익 모델**로 설계만 해두면 좋음

---

### 3-5. 찐팬 인증 페이지

- 목적: 매칭 시 “신뢰도” 보여주는 용도
- 기능
  - 굿즈 사진 업로드
    - 유니폼 / 목도리 / 응원봉 등
  - 직관 승/패 기록 기반 통계
    - 최소 n회 이상 직관 시 칭호 부여
    - 예:
      - 승률 80~100%: 🧚 **승리 요정**
      - 승률 40~60%: 😐 **평범한 관람러**
      - 승률 0~20%: 😭 **패배의 아이콘**
- 사용처
  - 프로필/게시글에 **뱃지/칭호**로 노출 → 동행 글에서 신뢰도 + 재미 요소

---

### 3-6. 직관 인증 페이지

- 인증 가능 시간
  - 경기 시작 1시간 전 ~ 당일 23:59
- 인증 방식
  - 티켓 화면 업로드 (사진 스캔-mvp에선 관리자 승인)
  - 위치 기반 인증 (경기장 반경 1km 내 체크인)
- 인증 성공 시
  - 해당 경기 직관 횟수 +1
  - 승/패 결과 반영 → 승률/칭호에 반영
  - 스탬프 형식으로 마이페이지에 누적

---

### 3-7. 마이페이지

- 내 정보
  - 프로필 (한 줄 소개, 사진)
  - my 응원팀 (MVP에서는 정관장 위주, 확장 시 중복 가능)
- 나의 기록
  - 직관 인증 리스트 (어떤 경기, 언제)
  - 찐팬 인증(굿즈, 칭호, 승률 등)
- 커뮤니케이션
  - 1:1 채팅 목록
  - 문의 (버그 제보, 신고 등)

---

### 3-8. 프로필 페이지 (상대 보기)

- 항목
  - 프로필 사진, 한 줄 소개, 나이대/성별
  - 응원팀
  - 성격/응원 스타일 태그
  - 찐팬 인증서 (굿즈, 직관 횟수, 승률) → **승률/직관수 카드 형태로 제공**
  - **사용자 신고 하기** (**New**)
  - 매너온도 (있다면)
- 동행 게시글에서 카드 클릭 → 이 페이지로 진입

---

### 3-9. 경기 종료 후 기능

- 경기 종료 후에 열리는 영역
  - 뒷풀이 모집
    - “경기 끝나고 ○○역 근처에서 맥주 드실 분”
  - 카풀
    - “○○역까지 가실 분 함께 택시”
  - MVP 선정
    - 이 경기의 MVP 선수 선정
  - 매너온도
    - 동행자 매너온도 부여

이 부분도 **MVP1에선 선택사항**이고

동행 매칭이 잘 되면 자연스럽게 확장하면 좋을 기능들.
